My Work

Automated Facial Recognition for Missing Persons Identification

In my recent role as a Machine Learning Engineer Intern at Atfal Mafkooda, I constructed an automated system to process and organize image data from API-sourced JSON files, handling around 2000 cases. This system was enhanced with a facial detection and matching pipeline I developed, utilizing MTCNN and VGG-Face within the DeepFace framework to identify missing individuals from unlabeled image databases. Evaluated performance thoroughly using Euclidean_l2 and Cosine metrics, demonstrating their efficacy in pinpointing initial matches.Deployed the system using Streamlit for interactive viewing of potential matches at various thresholds, significantly reducing manual photo reviews and accelerating the identification of missing individuals.
